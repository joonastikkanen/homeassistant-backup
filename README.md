# homeassistant-backup
Python script to backup Homeassistant to remote

## config.yaml

```yaml
ssh_user: "root"
ssh_password: "password"
ssh_host: ""
ssh_port: 22
homeassistant_url: "http://homeassistant.joonastikkanen.fi"
homeassistant_token: "token"
backup_dir: "/backup"
remote_backup_dir: "/backup"
```
